macOS menu bar app

LidStay 설치

Mac을 필요한 시간 동안 켜두고, 디스플레이 잠자기는 그대로 허용하는 가벼운 메뉴바 앱입니다.

LidStay 다운로드 GitHub 열기

가장 쉬운 설치

  1. `LidStay 다운로드`를 누릅니다.
  2. 압축을 풀고 `LidStay.app`을 Applications 폴더로 옮깁니다.
  3. 앱을 실행한 뒤 메뉴바의 눈꺼풀 아이콘을 누릅니다.

처음 실행할 때 macOS 보안 경고가 나오면 Finder에서 앱을 우클릭한 뒤 `열기`를 선택하세요.

터미널로 설치

git clone https://github.com/ghkdqhrbals/LidStay.git && cd LidStay && ./packaging/install-with-brew.sh
  1. 터미널에 명령어를 붙여넣습니다.
  2. 스크립트가 앱을 빌드하고 Homebrew cask로 설치합니다.

업데이트

cd LidStay && git pull && ./packaging/install-with-brew.sh

제거

brew uninstall --cask lidstay
GitHub 열기

LidStay는 공개 macOS 전원 API만 사용합니다. 덮개를 닫았을 때의 동작은 Mac 모델과 전원 상태에 따라 달라질 수 있습니다.
