LidStay
Release GitHub

macOS menu bar app

Mac을 필요한 동안만 켜두세요.

LidStay는 화면은 쉬게 두고, 다운로드나 개발 서버 같은 작업은 조용히 이어가게 해줍니다.

LidStay.pkg 다운로드 최신 버전 보기

pkg를 열고 안내대로 설치하면 메뉴바에 LidStay가 나타납니다.

메뉴바에서 바로 켜기, 중지, 시간 선택을 작은 메뉴에서 처리합니다.
화면은 쉬게 디스플레이 절전은 그대로 두고 필요한 작업만 유지합니다.
배터리 기준 설정 배터리가 낮아지면 자동으로 멈추도록 설정할 수 있습니다.
