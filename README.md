# LidStay

<img width="200" height="183" alt="image" src="https://github.com/user-attachments/assets/14374f3e-198e-450d-a55e-e25a074cb64e" />


The simplest way to keep your Mac awake with the lid closed.

LidStay is a tiny macOS menu bar app. One click keeps your Mac running, and battery mode is included.

Website: https://ghkdqhrbals.github.io/LidStay/

## Install

Download the latest installer:

```text
https://github.com/ghkdqhrbals/LidStay/releases/latest/download/LidStay.pkg
```

## Features

- One click to keep your Mac running with the lid closed.
- Works on battery when you need to keep moving.
- Display protection dims the built-in screen when possible.
- Choose a duration or keep it on until you stop it.
- Login launch, notifications, automatic updates, and in-app removal are available in Options.

LidStay uses public macOS power APIs only.
